# Panduan Penempatan Aset Branding (Hero & Kabupaten)

Dokumen ini menjelaskan di mana Anda harus meletakkan gambar-gambar ikonik untuk bagian Hero (puncak halaman) dan visualisasi umum Kabupaten Rembang.

## 1. Lokasi Folder
Seluruh aset branding harus diletakkan di:
`public/assets/branding/`

## 2. Daftar File yang Diperlukan

| Komponen | Nama File | Deskripsi |
| :--- | :--- | :--- |
| **Hero Background** (Beranda) | `Kabupaten Rembang Hero.jpg` | Background utama di bagian atas halaman Beranda. |
| **Kabupaten Preview** (Statistik) | `Kabupaten Rembang.jpg` | Gambar utama yang muncul di modal statistik sebelum memilih kecamatan. |

> [!IMPORTANT]
> - Gunakan format **.jpg**.
> - Pastikan resolusi gambar **Kabupaten Rembang Hero.jpg** cukup tinggi (minimal 1920x1080) agar terlihat tajam.
> - Pastikan penamaan file **persis sama** (termasuk spasi dan huruf kapital).

## Langkah Penerapan
1. Siapkan file gambar Anda.
2. Ubah namanya menjadi `Kabupaten Rembang Hero.jpg` dan `Kabupaten Rembang.jpg`.
3. Pindahkan ke folder `public/assets/branding/`.
4. Muat ulang halaman browser untuk melihat hasilnya.
